# 第7回課題

提出パターン：A-1

テーマ：
進化が速すぎる AI 分野で、毎日大量に流れてくるニュースを英語のまま追うのは面倒で、結局キャッチアップできず「追いついた気」になるだけ、というのを解決したい思いから作成しました

🔗 https://catchup-ai.vercel.app/

## v2からの差分

1点目：日々の読み進捗を可視化する「消化率バー（既読 / 全件・%）」と、既読・理解済みの記事を隠す「未読のみ」フィルタを追加し、毎日の積み残しを把握しながら読み進められるようにした

2点目：各記事の時刻を「3時間前」のような相対表示に変更（ホバーで絶対時刻を表示）し、ヘッダに最終更新時刻を表示することでニュースの鮮度が一目で分かるようにした

3点目：既読ステータスとメモの状態を Context で一元管理する構成に変更し、トップ記事カードと一覧カード間で状態を共有（壊れた保存データに対する耐性も追加）
